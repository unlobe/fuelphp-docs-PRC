欢迎来到 FuelPHP

FuelPHP 是个简单、灵活、社区驱动的 PHP 5 网站框架。它诞生于挫折的人与当前可用的框架和发展与社区的开发者的支持。FuelPHP非常便携,适用于几乎所有的服务器和自豪在清洁语法。

快速安装

如果你重视你的时间,想把事情在秒而不是分钟启动和运行,您可以使用我们的快速安装程序。使用 curl库,你可以安装一个非常基本的版本的Oil。接着便能建立一个全新的 Fuel 应用程序。

# quick install oil from the web
$ curl get.fuelphp.com/oil | sh
# now that oil is installed, create a blog project in a directory called Sites
$ cd Sites/
$ oil create blog

更多信息, 请查看 安装 页面.

手动安装

如果你无法快速的安装, 你也可以手动安装FuelPHP. 你能在 安装 页面找到详细的安装说明.
