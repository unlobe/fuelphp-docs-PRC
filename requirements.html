需求

Web 服务器
  • 任何 web 服务器. Fuel 已经在 Apache, IIS and Nginx通过测试.
  • 工作在主要的操作系统上包括*nix和Windows。
PHP 版本
  • 5.3.3或者更高版本.
PHP 扩展

Fuel使用PHP定义为“扩展”的功能，. 避免任何可能的兼容问题, Fuel只使用部分记录在 PHP 网站的代码的扩展. 如果使用扩展的话，那是部分的PECL库（这需要由系统管理员手动安装）， 如果存在Fuel将使用他们，不存在将不使用。

这是一个扩展列表，使用它们的类：

扩展 用途 注解
fileinfo 上传 文档.
请注意，在Windows平台上，需要手动安装一个DLL
mbstring n/a 文档.
如果不存在，没有多字节支持框架。你可以检查mbstring常看看你有多字节支持安装。
mcrypt Crypt 文档.
如果不存在，phpseclib是用来模拟其功能。
PHPUnit
  • PHPUnit 如果要运行单元测试，需要3.7或更大的版本。

建议

  • Web 服务器 - Apache 2.4+,运行在 *nix上.
  • mod_rewrite - 从URL中删除 index.php .
